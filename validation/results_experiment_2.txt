======================================================================

Hypothesis: Finding, Checking, Recognizing have distinct type signatures
This proves they are THREE DIFFERENT OPERATIONS

OPERATION SIGNATURES:
----------------------------------------------------------------------
1. Finding:
   Type:   Problem -> Candidate
   Input:  List[(List[int], int)]
   Output: Callable[[List[int]], int]

   Result: <built-in function sum>
   Python type: builtin_function_or_method

2. Checking:
   Type:   (Problem, Candidate) -> Boolean
   Input:  List[(List[int], int)], Callable
   Output: bool

   Result: True
   Python type: bool

3. Recognizing:
   Type:   (Problem, Candidate) -> Epistemic
   Input:  List[(List[int], int)], Callable
   Output: float  (confidence in [0,1])

   Result: 1.0
   Python type: float
----------------------------------------------------------------------

TYPE COMPARISON:
----------------------------------------------------------------------
Finding output:     builtin_function_or_method (Callable)
Checking output:    bool            (Boolean)
Recognizing output: float           (Epistemic/float)
